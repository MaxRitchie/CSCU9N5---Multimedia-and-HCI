Shortcuts

Here are tables showing some shortcuts that you may find helpful when using GIMP!

Toolbox Shortcuts
Rectangle Select R
Ellipse Select E
Free Select F
Scissors I
Eraser Shift + E
Pencil N
Paint Brush P


File Shortcuts
New Image Ctrl + N
Open Image Ctrl + O
Open Image as New Layer Ctrl + Alt + O
Save Image Ctrl + S
Save Under a New Name Ctrl + Shift + S
Duplicate Image Ctrl + D


Edit Shortcuts
Undo Ctrl + Z
Redo Ctrl + Y
Copy Ctrl + C
Cut Ctrl + X
Paste Ctrl + V
Paste as New Image Ctrl + Shift + V
